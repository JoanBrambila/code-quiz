## Description

The code-quiz was created to test the knowledge of JavaScript with this quiz. It is also a timed quiz, as soon as the timer gets to zero, your score will be displayed.

N/A

## Usage

Using this Website is very easy, as you open the page, you will see the Code-Quiz instructions and a start button, when you press start, a timer will start counting down and the questions will be displayed, when you answer the questions, the next one will be displayed, when the timer reaches zero, the quiz will end and the scores will be displayed.
